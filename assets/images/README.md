# Screenshots Directory

## Adding Screenshots

To add your app screenshots, save them in this directory with the following names:

- `metronome-screenshot-1.png` - Main Interface (BPM Control)
- `metronome-screenshot-2.png` - Presets Screen
- `metronome-screenshot-3.png` - Time Signature Selector
- `metronome-screenshot-4.png` - Subdivision Options
- `metronome-screenshot-5.png` - Easy to Use Interface
- `metronome-screenshot-6.png` - App Icon

## Image Requirements

- **Format**: PNG or JPG
- **Recommended size**: 750x1334px (iPhone screenshot size) or similar aspect ratio
- **Optimization**: Compress images for web to keep page load fast

## How to Add

1. Export screenshots from your iPhone/Simulator
2. Rename them according to the list above
3. Copy them to this `assets/images/` directory
4. The landing page will automatically display them

The page will show placeholders if images are missing, so you can add them gradually.
